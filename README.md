# Primer proyecto HTML

Este proyecto muestra mi aprendizaje en el HTML de forma 100% manual y sin el uso de inteligencia artificial. Quiero aprender los fundamentos necesarios para luego poder desarrollar mis propios sistemas y programas con ayuda de la IA pero siendo ahora capaz de guiarla y orientarla adecuadamente, entendiendo todo lo que hace y no trabajando a ciegas.

## Recetas de comida

En este caso en particular, se muestran las recetas para poder preparar algunos platos de comida, utilizando las nociones básicas de HTML para poder mostrar la información de manera adecuada.
